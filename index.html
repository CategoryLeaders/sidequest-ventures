// the model

How we work

AI compressed the cost of building software by 10x. We're using that.

The venture studio model — building multiple companies from a shared team and thesis — is now dramatically more capital-efficient. One senior team can ship what used to require dozens of engineers.

We don't invest in founders. We don't run a programme. We identify high-signal opportunities where AI changes the economics, and we build the companies ourselves. From zero to product, revenue, and operating independence.

Some sidequests are quick experiments. Others become the whole story. We won't know which until we build them.

01

Identify

Find high-signal opportunities where AI changes the economics of an industry or workflow. Research, validate, discard the bad ones fast.

02

Build

Design, develop, and launch from inside the studio. One team, shared infrastructure, fast iteration. Ship the first version in weeks, not months.

03

Scale

Install leadership, raise capital, and spin out into an independent operating company. The studio retains equity and stays close.
